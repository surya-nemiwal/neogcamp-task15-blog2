A simple way to center html tags.

Introduction

Have you ever used display : 'float' property to center the child elements of a div. Then you know how cumbersome it is to write many lines of code just to center some html tag!

What if now I tell you that you can do same task of centering a element with just one line of code. Don't believe me? read below and find out yourself!

We'll be using flexbox to center the content of a div. So before we look at the actual code lets first understand what flexbox is?

flexbox is a poweful css property to manage scaling and all the extra space within a html tag.

We'll be using flexbox to center the content of a div. So before we look at the actual code lets first understand what flexbox is?

flexbox is a poweful css property to manage scaling and all the extra space within a html tag.

The syntax to use flexbox is

        
        <div style="display: flex;">
            {child components}
        </div>

code-explatanation: To make a div flex-container i.e flexbox simply change the display property from block to flex. Isn't that simple

Properties of flexbox

There are many properties associated with flexbox like flex-grow, flex-wrap, justify-content, align-items.

But we will be using only two properties to center the child tags

  • justify-content : This property is used to align the content in main-axis. By default the value of justify-content is flex-start, and we have to change this value to "center" so as to center the child tags along the row i.e in horizontal direction

  • align-items : This property is used to align the content in cross-axis. By default the value of align-items is stretch, and we have to change this value to "center" so as to center the child tags along the column i.e in vertical direction

Now that we have basic understanding of how to declare and use flexbox lets write the code to center the child of a div tag

                
            <div style="display: flex; justify-content: 'center'; 
                align-items:'center' ">
                    {child component here ...}
            </div>

Tan-Tadan! thats it. Thats all you need to center the child tag i.e. to wrap the child tag within a div, whose display property is set to display: 'flex'; and
justify-content: 'center';align-items:'center'; .
